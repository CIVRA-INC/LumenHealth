# MVP Launch Checklist

A checklist template to guide the team through a safe and successful MVP launch.

---

## Pre-Launch

- [ ] All environment variables are set in production
- [ ] Database migrations have been run and verified
- [ ] Secrets have been rotated (API keys, tokens, passwords)
- [ ] Third-party service integrations tested in staging
- [ ] Error tracking and logging configured
- [ ] Backup strategy confirmed

## Launch Day

- [ ] Smoke tests passing against production environment
- [ ] Monitoring dashboards are live and showing expected metrics
- [ ] Alerting thresholds configured and tested
- [ ] Support channels are open and team is on standby
- [ ] Rollback plan documented and accessible
- [ ] Deployment completed and verified

## Post-Launch

- [ ] Metrics review at 1 hour post-launch
- [ ] Metrics review at 24 hours post-launch
- [ ] Incident runbook linked and accessible to all team members
- [ ] User feedback channels monitored
- [ ] Performance benchmarks recorded for baseline comparison
- [ ] Retrospective scheduled
